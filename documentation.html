Simple Tokenizer Documentation

This documentation explains how the Simple Tokenizer works, including mapping of characters to tokens, tokenizing strings, and decoding tokens back to text.

1. Overview

The Simple Tokenizer assigns a unique number (token) to each character. This allows text to be represented as a sequence of numbers for processing, and then converted back into characters.

2. Example Tokenizer Code

class SimpleTokenizer {
    constructor() {
        this.charToToken = {};
        this.tokenToChar = {};
        let tokenId = 1;

        // Lowercase letters (a=1, b=2, ...)
        for (let i = 0; i < 26; i++) {
            let ch = String.fromCharCode(97 + i);
            this.charToToken[ch] = tokenId;
            this.tokenToChar[tokenId] = ch;
            tokenId++;
        }
        // Numbers
        for (let i = 0; i < 10; i++) {
            let ch = String.fromCharCode(48 + i);
            this.charToToken[ch] = tokenId;
            this.tokenToChar[tokenId] = ch;
            tokenId++;
        }
    }

    tokenize(text) {
        return text.split('').map(c => this.charToToken[c] || 0);
    }

    decode(tokens) {
        return tokens.map(t => this.tokenToChar[t] || '?').join('');
    }
}
